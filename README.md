# Predicting F1 Pitstops

Kaggle Playground Series project focused on predicting Formula 1 pitstop outcomes using machine learning.

## Tools
- Python
- pandas
- scikit-learn
- XGBoost
- LightGBM
- CatBoost

## Structure

- `notebooks/` → experimentation and EDA
- `src/` → reusable code
- `submissions/` → Kaggle submission files
